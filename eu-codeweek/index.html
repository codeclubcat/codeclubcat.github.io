Code Club CAT @ EU Code Week

Guarda't la data: 7 - 22 d'octubre 2017

Et proposem dues activitats per la setmana de la Code Week a Europa!
Aprofita per conèixer els recursos de Code Club en Català amb una activitat diveritda per un grup de nens i nenes!

Juga, explora i aprèn amb Scratch!

Engega el teu Code Club!

Què farem?

  • Fes una activitat diferent: crea el teu vídeojoc!
  • Conèixes els nous projectes de Code Club?
  • Conèixes les targetes Scratch en Català
  • I més recursos per la difusió

Com ho farem?

  1. Nous projectes amb Scratch! Crea la teva banda de rock, i un videojoc sobre l'espai
  2. Targetes Scratch en català per aprendre ràpid noves funcionalitats de Scratch.
  3. Registra el teu Code Club i tindràs certificats, pòsters i més coses divertides!
  4. Posa't al mapa de la EU Code Week amb la teva activitat!
